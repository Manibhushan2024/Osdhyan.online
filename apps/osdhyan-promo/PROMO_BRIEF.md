# OSDHYAN Promo Brief

## Format

- Primary format: `1080x1920` vertical social ad
- Frame rate: `30 FPS`
- Duration: `34 seconds`
- Tone: premium, aspirational, disciplined, intelligent, result-driven

## Core Message

OSDHYAN is not just a mock-test app. It is a complete bilingual exam preparation ecosystem for Indian competitive exam aspirants, combining mock tests, PYQs, study materials, syllabus tracking, analytics, focus tools, and AI solution guidance in one system.

## Scene Order

1. Hero: Position OSDHYAN as the serious-preparation operating system.
2. Ecosystem: Show that the platform covers more than tests.
3. Test flow: Highlight English/Hindi switching, timed mocks, negative marking, and resume support.
4. Analytics: Show performance review, weak-area tracking, and AI explanation value.
5. Institute layer: Show question bank and content operations for coaching teams.
6. Brand close: Land the platform promise and target exam categories.

## Voiceover

1. Serious exam preparation needs more than random practice.
2. Meet OSDHYAN, the bilingual preparation ecosystem built for aspirants who want discipline, clarity, and performance.
3. Attempt full mock tests, switch between English and Hindi, solve previous year papers, and study from structured materials in one place.
4. Track your accuracy, spot weak areas, and use AI-powered explanations to learn from every mistake.
5. For institutes, OSDHYAN brings question banks, bilingual workflows, test creation, and content operations into one modern system.
6. OSDHYAN. Prep smart. Stay consistent. Perform with intent.

## Suggested Music Direction

- Genre: cinematic electronic with disciplined percussion
- Tempo: medium-fast, rising energy
- Feel: premium, focused, confident
- Avoid: playful startup music, comedy pacing, motivational cliches

## Render Commands

```bash
npm install --workspace apps/osdhyan-promo
npm run promo:dev
npm run promo:thumbnail
npm run promo:render
```
